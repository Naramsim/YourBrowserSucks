YourBrowserSucks.js.

This library allows you to gracefully inform your site visitors if their browser is outdated. It is a quick and dirty library to include in every web-page that tests the browser. If the tests do not pass the web-page will be overridded by YourBrowserSucks web-page. Something similar to this ▾

Tests

To check if your browser is good or a badass YourBrowserSucks checks if supports some CSS3 properties, if some JS element are usable, if WebGL and LocalSotrage are present and other few things like ´const´ and ´let´.

What if your browser sucks..?

Four links will be provided to download your favorite browser latest release. Only chrome will be suggested if you are on mobile.
